How it Works:

  • Enter the name of each player in turn and click on the Add Player! button.
  • Click on Create Teams! The app will select two teams.
  • Add match scores using the Add Match Result! button. The app will endeavour to create more evenly matched teams the next time you hit Create Teams!
  • Add or remove players using the appropriate buttons.
  • NB!! WARNING!! Refreshing the page will reset the app including removing all existing players and scores!!
